# 千易云招聘网站爬虫


----------

软件说明
-

----------
- **功能:**
爬取各大招聘网站招聘信息，并在当前脚本目录生成Excel表格与HTML静态页面
- **语言:**
Python 3
- **第三方包支持:**
pandas
xlwt
codecs
